# Go-Muktinath – Travel Guide Website

A fully bilingual (English/Nepali) single‑page travel guide for pilgrims and tourists planning a trip to Muktinath, Nepal. The site provides detailed itineraries, budget estimation, interactive maps, weather forecasts, and direct booking links – all with a clean, responsive design.



##  Features

- **Bilingual Support** – Toggle between English and Nepali with a single click. All static and dynamic content (including itineraries, FAQ, and map markers) is translated.
- **Interactive Map** – Leaflet map showing routes from Pokhara to Jomsom and Jomsom to Muktinath, with clickable markers for key locations.
- **Itinerary Planner** – Choose from five travel plans (Fast, Standard, Family, Adventure, Budget). Adjust the number of adults/children to see real‑time cost estimates.
- ** Budget Table** – Estimated costs for transport, accommodation, food, and permits.
- **FAQ Accordion** – Answers to common questions (bilingual).
- **7‑Day Weather Forecast** – Current conditions plus a 5‑day forecast (using OpenWeatherMap). Click any day for detailed info (wind, humidity, pressure, etc.).
- **Direct Booking** – Transport and hotel booking buttons link to partner websites (`muktinathdarshantours.com` and `hotelcrystalfall.com`).
- **Responsive Design** – Built with Bootstrap 5, works on desktop, tablet, and mobile.
- **Custom Styling** – Clean typography, hover effects, and a professional favicon set.

## Technologies Used

- HTML5, CSS3, JavaScript (ES6)
- [Bootstrap 5](https://getbootstrap.com/) – Layout and components
- [Leaflet](https://leafletjs.com/) – Interactive maps
- [OpenWeatherMap API](https://openweathermap.org/) – Weather data
- Custom i18n system – JSON‑based translations with dynamic placeholder replacement
